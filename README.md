# 🏡 StayFinder – Airbnb Clone Listing Platform

StayFinder is a full-stack web application inspired by Airbnb that allows users to explore, review, and manage property listings. Users can browse beautiful places, add their own listings, and share reviews.

---

## 🚀 Live Demo

🔗 https://major-project1-2yuh.onrender.com/

---

## 📌 Features

* 🔐 User Authentication (Login / Register)
* 🏠 Add, Edit, Delete Listings
* 📍 Location-based Listings
* ⭐ Reviews & Ratings System
* 📸 Image Upload for Listings
* 🌐 Fully Deployed on Render
* 📱 Responsive Design (Mobile Friendly)

## ⚙️ Installation & Setup

1. Clone the repository

```bash
git clone https://github.com/your-username/stayfinder.git
```

2. Navigate to project folder

```bash
cd stayfinder
```

3. Install dependencies

```bash
npm install
```

4. Setup environment variables
   Create a `.env` file and add:

```
MONGO_URI=your_mongodb_connection
CLOUDINARY_KEY=your_key
CLOUDINARY_SECRET=your_secret
SESSION_SECRET=your_secret
```

5. Run the server

```bash
npm start
```

6. Open in browser

```
http://localhost:3000
```
